Bassin 7

website

Bassin 7 is an area around Aarhus harbor with facilities such as saltwater swimming pool, dining areas etc. The website is developed as an information side which describes what Bassin 7 has to offer, including opening hours, special offers and other relevant information.


The website is a responsive solution, which is developed in HTML, CSS and Javascript.

My Plants

web app

My plants is a mobile web app, that helps the user take care of their plants. On the web app the user can add the different plants you have in your home, to their personal profile page.


My plants are developed as a single page application, with the use of HTML, CSS and Javascript. Furthermore, has there been used firebase for login authentication and storage.
